# Getting and Cleaning Data - Code Book

By running `run_analysis.R` (mentioned in README.MD) will create a new datafile `tidy.txt`.
The dataset contains data of subjects that performed certain activities and there corresponding measurements.

## Activities Performed

* `WALKING`
* `WALKING_UPSTAIRS`
* `WALKING_DOWNSTAIRS`
* `SITTING`
* `STANDING`
* `LAYING`

## Identifiers

* `Subject` - The identifier of the test subject
* `Activity` - The activity performed by the subject.
